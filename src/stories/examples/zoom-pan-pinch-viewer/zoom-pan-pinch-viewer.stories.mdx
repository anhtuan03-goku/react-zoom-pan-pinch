# ZoomPanPinchViewer

Component bọc `TransformWrapper` + `TransformComponent`, kèm lớp SVG (`viewBox` khớp ảnh) và tùy chọn **click path → `zoomToElement`** (animation).

### Preview:

  <Story name="Ảnh + hotspot SVG">{(args) => <Template {...args} />}</Story>
</Canvas>

## Ghi chú

- `svgOverlay.viewBox` nên trùng với kích thước pixel gốc của ảnh (ví dụ `0 0 ${naturalWidth} ${naturalHeight}`).
- Mỗi hotspot cần `id` duy nhất; dùng `overlayPathIdPrefix` nếu có nhiều viewer trên một trang.
